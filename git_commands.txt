GIT COMMANDS

I. First step

1) git init  (for initialization of the repository)
2) git add .   (. means any file extension)
  git add *.java (all files only with java extensions)
3) git commit
  git commit -m "pass some message"
4) git remote add origin <remote url>
Eg: git remote add origin https://github.com/SharanyaKishtapuram-star/VISEM_JFS.git
5) git push origin master(old version)
  git push origin main(new version)
6) you can heck it in remote repository

DAY 2


dir /a:h 
  
1.To remove a empty directory  - rmdir directory_name

2.To remove a directory with content(such as text files in it) - rmdir /s /q directory_name

3.to make a directory - mkdir priscilla

4. to hide a directory - attrib +h priscilla

5. to unhide a directory - attrib -h priscilla

6. to display all the folders or directories - dir

7. to rename a directory - rename previous_name current_name
rename priscilla pris

8. to display the content of a file - type file_name

9. to display a hidden folder - dir /a:h

10. to create a new text file - notepad filename.txt

11. to forward content from one file to another file
create a file first : notepad priscilla.txt
enter some content in it hello jfs
move the content to another file
type priscilla.txt>pris.txt (> is used for forwarding the content)

the pris.txt will be created automatically and the content will be moved to it

12. to delete a file - del <filename>

13. to delete all text files - del *.txt

14. the list of wild characters:
 
 ? - single character
 * - 0 or more characters
 
15. to display all single character named file with extension .txt:dir?.txt

16. to display all files with extension
